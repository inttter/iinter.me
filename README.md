# iinter.me
    
* [**Next.js 14**](https://nextjs.org)
* [**Vercel**](https://vercel.com)
* [**Tailwind CSS**](https://tailwindcss.com)
* [**Vercel Analytics**](https://vercel.com/analytics)

## Run locally

```bash
git clone https://github.com/inttter/iinter.me.git
cd iinter.me
npm install
npm run dev
```
