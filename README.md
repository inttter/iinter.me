# iinter.me
    
* [**Next.js**](https://nextjs.org)
* [**Tailwind CSS**](https://tailwindcss.com)

## Clone locally

```bash
git clone https://github.com/inttter/iinter.me.git
cd iinter.me
npm install
npm run dev
```

To update a post's last updated time, run:

```bash
npm run last-updated -- content/[file].md
```
